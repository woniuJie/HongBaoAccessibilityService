# HongBaoAccessibilityService
通过AccessibilityService 来实现 微信抢红包的功能

## 基本思路：
1、监测通知栏，如果带有 [微信红包] 字样的通知到后，点击进入到详情页
2、监测当前页面有没有 微信红包 的 组件，如果有点击
3、点击后进入到开红包页面，点击 開 的按钮
